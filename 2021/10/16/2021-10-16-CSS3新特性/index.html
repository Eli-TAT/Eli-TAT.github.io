记录 CSS3 一些新特性及简单用法

1. flex 弹性盒子布局

详细使用可参考Flex 布局教程：语法篇、flex 布局的基本概念

简单来说，设置了display:flex或者display:inline-flex的元素就成为了一个 flex 容器，flex 容器中的直接子元素被称为 flex 项目（flex item），简称项目。在flex 容器中有水平、竖直方向两条轴线，水平轴为主轴，竖直轴为交叉轴，它们是 flex 布局的核心，通过设置相关属性就可以对 flex 容器中的项目进行布局操作。本质上是一种一维布局方式。

flex 容器的属性
设置在 flex 容器上的属性

  • flex-direction
    定义了容器中 flex 项目沿轴线的排列方式，实际上是在修改主轴的位置和方向
    属性值: row(默认值，从左到右水平排列)｜ row-reverse ｜ column ｜ column-reverse
  • flex-wrap
    定义了 flex 项目占满一行后的的换行方式
    属性值: nowrap(默认值，不换行)｜ wrap(换行，第一行在上方)｜ wrap-reverse(换行，第一行在下方)
  • flex-flow
    属于 flex-direction 和 flex-wrap 属性的简写
    属性值: row nowrap(默认值) ｜…
  • justify-content
    定义了 flex 项目在主轴上的对齐方式
    属性值: flex-start（默认值） ｜ flex-end ｜ center ｜ space-between ｜ space-around
  • align-items
    定义了 flex 项目在交叉轴上的对齐方式
    属性值: flex-start ｜ flex-end ｜ center ｜ baseline(项目中第一行文字的基线对齐)｜ strech(默认值，项目高度未设置或者为 auto 时，将填满整个容器的高度)
  • align-content
    定义了多根轴线的对齐方式，如果项目只有一根轴线，该属性不起作用。

flex 项目的属性
设置在 flex 项目上的属性

  • flex-grow
    定义了 flex 项目放大的比例。接收一个无单位数<number>，默认值为1。
  • flex-shrink
    定义了 flex 项目缩小的比例。接收一个无单位数<number>，默认值为1。
  • flex-basis
    定义了 flex 项目在主轴上的初始尺寸。接受一个有效的宽度值，默认值为0。
    当一个元素同时被设置了 flex-basis (除值为 auto 外) 和 width (或者在 flex-direction: column 情况下设置了height) , flex-basis 具有更高的优先级。
  • flex
    前三个属性的简写flex:flex-grow flex-shrink flex-basis;
    可以使用一个，两个或三个值来指定 flex属性：
    （1）单值语法
    - 值为一个无单位数，会被当作flex:<number> 1 0;
    - 值为一个有效宽度值，会被当作flex:0 1 width;
    - 值为关键字，none、auto或initial。
    （2）双值语法
    第一个值必须为一个无单位数，它会被当作flex-grow>的值。第二个值必须为以下之一：
    - 一个无单位数：它会被当作flex-shrink的值
    - 一个有效的宽度值: 它会被当作 flex-basis的值
    （3）三值语法
    略…
    其中 flex:1; 这种简写方式比较常用
  • align-self
    允许单个flex项目在交叉轴上有自己的对齐方式，会覆盖已设置的 ```align-items``。
  • order
    接收一个整数<integer>,规定了flex容器中的flex项目在布局时的顺序。仅仅改变了元素的视觉顺序，并不会影响元素的逻辑顺序。

2. grid 网格布局

类似的，设置了display:grid或者display:inline-grid的元素就成为了一个 grid 容器，其中的直接子元素被称为gird项目，通过设置行和列将grid容器划分出单元格，再将grid项目放在单元格中进行布局。这是一种二维的布局方式。

grid 容器的属性
设置在 grid容器上的属性：

  • grid-template-rows
    设置grid容器中的行
  • grid-template-columns
    设置grid容器中的列
  • grid-auto-flow
    定义单元格在容器中排列的方向
  • justify-content
    定义grid容器中单元格，在水平方向的对齐方式
  • align-content
    定义grid容器中单元格，在竖直方向的对齐方式
  • justify-items
    定义单元格中项目，在水平方向的对齐方式
  • align-items
    定义单元格中项目，在竖直方向的对齐方式
  • grid-auto-rows
  • grid-auto-columns

grid 项目的属性
设置在 grid项目上的属性：

  • grid-row
    定义单个grid项目所占的行，grid-row-start与grid-row-end的简写
  • grid-column
    定义单个grid项目所占的列，grid-column-start与grid-column-end的简写
  • justify-self
    定义单元格中单个项目，在水平方向的对齐方式
  • align-self
    定义单元格中单个项目，在竖直方向的对齐方式

3. CSS3 动画

CSS3 中动画的使用只需要两步：

  1. 定义动画
    通过@keyframs 关键字声明一个 CSS3 动画，通过百分比设置关键帧，并在每一帧进行相应的操作。
    特别的，如果动画只有 0%和 100%两个关键帧，可以用 from、to 关键字替代。
  1. 引用动画
    定义好动画后，在需要的地方使用animation相关属性进行引用即可。
    常用属性
  • animation-name 动画名
  • animation-duration 动画持续时间
  • animation-timing-function 动画播放的速度函数
  • animation-delay 动画延迟时间
  • animation-iteration-count 动画播放次数

使用简写属性会更方便：
animation: 动画名 时间 函数 延迟 次数;

4. 过渡

CSS3中过渡的使用

常用属性

  • transition-property 需要应用过渡的属性
  • transition-duration 过渡持续时间
  • transition-timing-function 过渡的时间函数
  • transition-delay 过渡延迟时间

简写
transition: 属性 时间 函数 延迟;
