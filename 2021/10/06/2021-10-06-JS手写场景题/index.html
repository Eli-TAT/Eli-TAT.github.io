汇总一些 JS 手写题，偏向于解决特定场景下的问题。

1. 数组扁平化

分别用迭代和递归的思想实现数组扁平化。

  • 迭代
  • 递归

其实 Array 类型提供了一个数组扁平化的 API，指定层数为 Infinity 即可将数组完全扁平化

2. 实现 sleep 函数

  • promise 风格的实现
  • async 风格的实现

3. 将虚拟 DOM 转换为真实 DOM

遍历 JS 对象模拟的虚拟 DOM 进行解析，创建元素节点，并为元素节点添加属性节点、文本节点，并以同样的方法处理子节点。由于 DOM 是层层嵌套的结构，因此应该使用递归的思想。

4. 实现一个函数，可以对 js 中各种类型的数据进行值复制

重点在于判断 js 数据类型，再根据不同的数据类型拷贝不同格式的数据（PS：注意区分引用类型的浅拷贝和深拷贝）

5. LRU 算法

LRU（Least recently used，最近最少使用）算法根据数据的历史访问记录来淘汰数据，核心思想是“如果数据最近被访问过，那么将来被访问的几率也更高”。LRU 是一种内存数据淘汰策略，常用于当内存不足时淘汰最近最少使用的数据。

最常见的实现是使用一个链表保存缓存数据，为了简单起见，可以考虑使用 Map 类型（Map 对象在迭代时会根据对象中元素的插入顺序来进行）

6. 实现一个简单的 jsonp

jsonp 利用了 script 标签，只需在请求的 api 的 src 后加上一个 callback 参数，并设置一个用于处理服务器响应的回调函数即可。可以这样理解，浏览器可以通过 script 执行 js 文件，jsonp 只是在请求到服务器数据后，将数据传递给指定的回调函数去进行处理。

7. 使用 XHR 手写一个 AJAX（用 Promise 封装）

AJAX 全名 Asynchronous JavaScript and XML（异步 JavaScript 和 XML），它是一种使用现有标准的新方法，通过它可以实现在不刷新页面的情况下，与服务器交互获取数据来更新页面的部分内容。AJAX 通过 XMLHttpRequest 对象实现，XHR 对象用于与服务器交互，它可以在不刷新页面的情况下请求特定 URL，获取数据，这就允许网页在不影响用户操作的情况下，更新页面的局部内容。

XHR 的基本使用：

  1. 通过 XMLHttpRequest()构造函数初始化一个 XHR 实例对象
  2. 通过 open(method,url,async)方法初始化一个请求
  3. 通过 setRequestHeader()设置请求头
  4. 通过 send(body)发送请求，body 为请求体
  5. 通过 onreadystatechange 监听 readystate 属性值的变化。该属性代表请求的状态码（0：未初始化请求；1：open()已调用；2：send()已调用；3：响应数据下载中；4：响应数据下载完成）
  6. 响应数据下载完成后，通过 status 属性判断请求的响应状态，对响应作出处理，status 即 HTTP(S)的响应状态码
    有关 XHR 的详细用法请参考 MDN

8. 匹配括号问题

  • 利用正则表达式

采用“从内部开始瓦解”的思想，将一对相邻且匹配的括号替换成空字符串，从内部逐层开始匹配。此处提供的示例代码只能匹配纯括号字符串。

  • 利用栈数据结构

遍历字符串，碰到左括号就入栈，碰到右括号则将栈顶元素弹出，比较左右括号是否匹配。如果不匹配，则返回 false，否则直到遍历完整个字符串，判断栈是否已清空，未清空则返回 false。
