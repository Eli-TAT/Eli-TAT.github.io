1. 水平居中

水平居中很容易实现，只需一条代码就能搞定。

  • 文本水平居中
  • 块级元素水平居中

2. 水平垂直居中

难点在于实现垂直居中，实现的方式比较多，需要根据实际情况进行选择。

2.1 文本水平垂直居中

2.1.1 单行文本

设置 line-height 与父容器的 height 相等即可实现垂直居中效果。

2.1.2 多行文本

  • 父级元素高度不固定

面对父级元素高度不固定的情况，容器高度只能通过内部文本撑开，设置padding-top 和 padding-bottom 相等即可实现垂直居中效果。

  • 父级元素高度固定

为父容器设置display:table-cell;vertical-align:middle，使之具有表格单元格元素的特性，并使内容垂直居中即可。
vertical-align 只对行内元素、表格单元格元素生效，不能用于垂直对齐块级元素。vertical-align 用法

IE7 及以下版本并不能很好的支持 display:table 和 display:table-cell ，可以考虑用 css hack 解决。具体可参考 https://blog.csdn.net/freshlover/article/details/12132801

2.2 块级容器居中

2.2.1 子 div 大小固定

子 div 宽高固定时，可以设定水平和垂直偏移(相对定位父容器宽高的 50%)，再将子元素向上和向左作负向 margin(子 div 的实际宽高一半大小的距离)。该方法兼容 IE7、IE6。

2.2.2 子 div 大小不固定

  • 利用 transform 属性的 translate

相较上面子 div 大小固定的情况，这里不修改子元素的 margin 值，而是使用 transform 属性的 translate 进行偏移。由于 transform 属于 css3 属性，所以会有浏览器兼容问题，不支持 IE9 以下版本。

  • 利用绝对布局 absolute

绝对定位元素不在普通文档流中渲染，因此 margin:auto 可以使内容在通过 top: 0; left: 0; bottom: 0;right: 0; 设置的边界内垂直水平居中。该方法不兼容 IE7、IE6。

  • 利用 vertical-align

为父容器设置display:table-cell;vertical-align:middle，使之具有表格单元格元素的特性，并使内容垂直居中。
